#Windows Github
###Update 2016-12-12

###SSH keys Setting
1. 生成Key
C:\Users\Administrator>ssh-keygen -t rsa  
默认id_rsa.pub目录：C:\Users\Administrator\.ssh
2. 复制id_rsa.pub的内容到Github账户下  
Settings-->SSH and GPG keys--New SSH key
